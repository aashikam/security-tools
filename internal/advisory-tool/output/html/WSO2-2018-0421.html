OVERVIEW
It has been identified that the ML application sends some sensitive information in a GET request where the data is trasnferred in query parameter.
DESCRIPTION
ML application login page of DAS 3.1.0, sends credentials in GET request as query paramters in plain text.
IMPACT
When the sensitive information is passed in query parameters, those can get exposed via server logs, browser history etc.
SOLUTION
The recommended solution is to apply the provided patch/update to the affected versions of the products.
